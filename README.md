# Glory.report

Destiny 2 PvP tools, including player encounters, daily reports, and more.  
Built with [Nuxt](https://nuxt.com), and [Tailwind CSS](https://tailwindcss.com).

## How to run

```bash
git clone git@github.com:julesrx/glory.report.git
cd glory.report
pnpm install
pnpm dev
```

## How to contribute

```bash
# fork it
# clone it
# dev it
pnpm lint
# make a pull request
```
